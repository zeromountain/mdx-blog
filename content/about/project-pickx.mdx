## 프로젝트 소개

pickX.AI는 이미지 생성 AI 모델을 활용한 사내 이미지 생성 서비스입니다.

## 사용 기술

React, Next.js, Typescript, Tanstack Query, Chakra UI, zustand

## 주요 기여

### Tanstack Query 적용

기존 axios를 사용한 API 호출 방식에서 Tanstack Query로 변경하여 재사용성이 떨어지는 axios 템플릿 코드를 개선했습니다. 서버 상태 사용으로 클라이언트 상태 관리를 최소화하고 API 호출을 개선했습니다.

### zustand 적용

기존 useState를 사용한 상태 관리 방식에서 zustand로 변경하여 지역 상태 관리 코드를 전역 상태 관리로 클라이언트 상태 관리를 통합하고 개선했습니다.

### 프로젝트 폴더 구조 개선

기존 프로젝트 폴더 구조에서 모듈화된 폴더 구조로 변경하여 프로젝트 관리 및 유지보수 용이성을 개선했습니다.

### 이미지 리스트 페이지 개선

이미지 레이아웃 시프트, LCP, FCP 개선을 통해 이미지 메타데이터에서 추출한 width, height 정보를 통해 이미지 레이아웃 시프트를 개선했습니다. lazy 속성을 사용해 효율적인 이미지 자원의 로드 관리로 이미지 로드 속도를 개선하고(평균 100ms -> 평균 200ms), 최신 이미지 포맷(png -> webp/avif)을 적용했습니다.

### CDN 도입

기존 S3 버킷에 CloudFront 도입 및 이미지 리사이징 최적화를 건의하여 이미지 리사이징 최적화로 이미지 로드 속도를 개선했습니다(평균 200ms -> 평균 50ms).
