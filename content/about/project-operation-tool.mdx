## 프로젝트 소개

Four Ground Nine 게임 운영 툴입니다.

## 사용 기술

React, Next.js, Refine, Typescript, Tanstack Query, Tanstack Table, MUI, Tailwind CSS, zustand, react-hook-form

## 주요 기여

### Refine 적용

기존 프로젝트 폴더 구조에서 Refine 프레임워크를 적용하여 프로젝트 관리 및 유지보수 용이성을 개선했습니다.

### 도커 컨테이너 적용

도커 컨테이너 이미지 사이즈를 최적화하여 빌드 속도를 개선했습니다.

### 멀티 도메인 적용

provider 추가로 멀티 도메인을 적용하여 다수의 API 도메인 관리 용이성을 개선했습니다.

### 테이블 컴포넌트 개선

Tanstack Table 적용으로 테이블 컴포넌트를 개선하여 서버 상태와 동일한 테이블 컴포넌트 구현으로 비동기 데이터 처리 용이성을 개선했습니다.

### 다국어 지원

i18next 적용으로 다국어를 지원하여 영/중/일 언어 지원으로 어드민 사용자 편의성을 개선했습니다.
