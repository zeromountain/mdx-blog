## 프로젝트 소개

nextjs app router를 사용한 블로그 서비스입니다.

## 사용 기술

Next.js, Typescript, Tailwind CSS, NextUI, notion API

## 프로젝트 링크

[https://zeromountain.pro](https://zeromountain.pro)

## 주요 기여

### 블로그 포스트 메타데이터 추가

generateMetadata 함수와 generateStaticParams 함수를 추가하여 블로그 포스트 메타데이터를 추가함으로써 검색엔진 최적화 및 소셜 미디어 공유 시 메타데이터를 제공했습니다.

### ISR 적용

Incremental Static Regeneration을 적용하여 블로그 포스트 캐싱을 최적화하고, 블로그 포스트를 SSR 대신 ISR로 캐싱하여 블로그 포스트 로드 속도를 개선했습니다.
