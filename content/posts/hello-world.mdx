# Hello World

이것은 contentlayer를 사용한 첫 번째 블로그 포스트입니다.

## 마크다운 기능

마크다운에서는 다양한 기능을 사용할 수 있습니다:

- **굵게** 표시하기
- _기울임체_ 사용하기
- ~~취소선~~ 표시하기
- [링크](https://example.com) 추가하기

### 코드 블록

```javascript
function greeting(name) {
  return `안녕하세요, ${name}님!`;
}

console.log(greeting('방문자'));
```
